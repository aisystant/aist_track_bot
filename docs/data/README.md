# Данные

> Модель данных бота. Описывает **какие** данные хранятся и собираются.

---

## Структура раздела

| Документ | Описание |
|----------|----------|
| [tables.md](tables.md) | Все таблицы БД и их поля |
| [metrics.md](metrics.md) | Метрики, их источники и формулы расчёта |

---

## Связь с процессами

- **Процессы** описывают *как* данные собираются
- **Данные** описывают *что* хранится

Пример:
- Процесс `P-01 Отслеживание активности` → записывает в таблицу `activity_log`
- Таблица `activity_log` описана в `tables.md`

---

## Принципы

1. **Единый источник истины** — структура таблиц описана здесь, код должен соответствовать
2. **Версионирование** — изменения в схеме данных требуют обновления документации
3. **Трассируемость** — для каждой метрики указан источник данных
