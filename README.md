# AIST Track Bot — AI-помощник по системному развитию

Telegram-бот [@aist_track_bot](https://t.me/aist_track_bot) с персонализированным обучением от [Мастерской инженеров-менеджеров](https://system-school.ru/).

**Миссия**: Помочь пользователям трансформироваться из случайных учеников в систематических — собранных и удерживающих внимание на своём развитии.

---

## Ключевые понятия

> Полная терминология: **[docs/ontology.md](docs/ontology.md)**

| Термин | Определение |
|--------|-------------|
| **Ученик** | Пользователь, проходящий Марафон |
| **Читатель** | Пользователь, использующий Ленту |
| **Марафон** | 14-дневная программа по системному развитию |
| **Лента** | Ежедневные дайджесты по выбранным темам в своём ритме |
| **Урок** | Единица теории в Марафоне (текст + вопрос) |
| **Задание** | Единица практики в Марафоне (инструкция + рабочий продукт) |
| **Дайджест** | Ежедневный материал в Ленте |
| **Фиксация** | Личный вывод Читателя после дайджеста |

---

## Два режима обучения

| Режим | Для кого | Как работает |
|-------|----------|--------------|
| **Марафон** | Хочет пройти курс последовательно | 14 дней: урок → вопрос → задание → рабочий продукт |
| **Лента** | Предпочитает свой темп | Выбор тем на неделю → ежедневные дайджесты → фиксация |

---

## Быстрый старт

1. Найдите [@aist_track_bot](https://t.me/aist_track_bot) в Telegram
2. Напишите `/start`
3. Пройдите онбординг (~2 минуты)
4. Выберите режим: Марафон или Лента

### Команды

| Команда | Описание |
|---------|----------|
| `/start` | Начать работу |
| `/learn` | Получить следующий урок/дайджест |
| `/feed` | Перейти в режим Лента |
| `/progress` | Посмотреть прогресс |
| `/profile` | Показать профиль |
| `/update` | Обновить настройки |
| `/help` | Справка |

## Для разработчиков

### Архитектура

```
Telegram ←→ Bot (Python/aiogram) ←→ PostgreSQL
                    ↓
              Claude API (генерация контента)
```

### Технологии

- Python 3.11 + aiogram 3.x
- Anthropic Claude API
- PostgreSQL
- Railway / Render

### Запуск

```bash
git clone https://github.com/aisystant/aist_track_bot
cd aist_track_bot
python -m venv venv && source venv/bin/activate
pip install -r requirements.txt
cp .env.example .env  # Заполнить токены
python bot.py
```

### Переменные окружения

| Переменная | Описание |
|------------|----------|
| `TELEGRAM_BOT_TOKEN` | Токен от @BotFather |
| `ANTHROPIC_API_KEY` | Ключ API Anthropic |
| `DATABASE_URL` | URL PostgreSQL |

### Структура проекта

```
aist_track_bot/
├── bot.py                    # Основной код (Марафон)
├── engines/feed/             # Режим Лента
├── db/                       # Модели и запросы БД
├── config/                   # Настройки
├── docs/
│   └── ontology.md           # Терминология (источник истины)
├── knowledge_structure.yaml  # Структура тем
└── CLAUDE.md                 # Инструкции для ИИ
```
